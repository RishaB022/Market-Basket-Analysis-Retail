# Market Basket Analysis – Retail

## Project Overview
This project performs an end-to-end Market Basket Analysis on the UCI Online Retail dataset to uncover customer purchasing patterns and cross-sell opportunities.

## Tools & Technologies
- Python (Pandas, mlxtend – Apriori & Association Rules)
- SQL (MySQL)
- Power BI (Interactive dashboards)

## Key Objectives
- Analyse customer basket behaviour
- Identify frequently purchased products
- Discover product pairings using association rules
- Visualise business insights using Power BI

## Files in this Repository
- **uci retail II MBA.ipynb** – Data cleaning, transformation, and association rule mining in Python
- **mba_sql.sql** – SQL queries for aggregation, basket analysis, and Power BI-ready views
- **mba_retail_pb.pbix** – Power BI dashboard with retail overview and cross-sell insights

## Key Insights
- Average basket size remains stable while revenue varies by product mix
- Strong seasonality observed with revenue peaks in Q4
- Certain product pairs show high confidence and lift, indicating strong cross-sell potential
- Significant variation in basket behaviour across countries
